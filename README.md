# fuckit

> 鼻（窦）炎患者康复手册。

本文主要谈及自己在鼻（窦）炎方面的遭遇以及救治办法，有同样苦恼的小伙伴可以一起维护本仓库，争取，让大家能够再次吸入一份清新的空气，做一个“好人”。

我个人是 16 年冬天开始出现鼻（窦）炎，此后一直在抗争。做过 CT，显示左上颌鼻窦发炎，做过两次穿刺（希望大家可以不用做这个玩意儿，现在都还有阴影）。年底在某个三甲医院检查，建议我做手术被我委婉拒绝了。期间，某天晚上，头痛的要死，一晚上没有睡觉，整个人都绝望了。此后就是各种吃药（中药也试过）各种难熬，唯独某个老中医配的西药很管用，但依赖性太强，偶尔吃一下。期间还打过几针输过几次液，都以失败告终。最近又去医院检查，做了鼻内镜，发现有鼻息肉，但医生不建议做手术（说是不到万不得已再做吧），让我尝试中医疗法，婉拒。直到最近几天，鼻塞的厉害，再次开始寻找解决办法，于是成文。

总之，这是一场长久战役，别放弃。

## 症状

鼻炎和鼻窦炎患者会有一些症状是重叠的，我也没办法完全分清，但总的来说以下几点还是鼻窦炎比较常见的：

- 鼻内异味，只有自己可以闻得到的腥臭味儿
- 严重鼻塞，吃东西如果不用嘴呼气我会被自己憋死😭，晚上睡觉也无法用鼻子呼吸，早上醒来口腔干的特别难受
- 眼部不适，通常伴随着发痒等症状
- 偏头痛（隐隐作痛）
- 流涕不止，反流等（清色和黄绿色的有区别哦）

还有其他一系列的附带影响，比如睡眠质量下降，记忆力减退，耳部病变等。

## 禁忌

- 辛辣
- 冰冷
- 熬夜
- 着凉
- 休息不够
- 拒绝运动

## 手术治疗

不到万不得已不要做？这个还是看个人取舍吧，我无法给出意见。但就我个人而言，我可能真的会选择做手术，但目前还没有做的打算。

- 鼻窦引流
- 息肉切除
- 鼻窦穿刺
  - 这个还是算了吧，我 16 年做了两次，至今还阴影未散。从你的鼻孔插入一根笔芯那么粗的钢针，穿透鼻挺（靠近眼角那个位置的）脆骨，到达窦腔，然后注射消炎液体，将窦腔内的炎症粘液用消炎液体冲洗干净，冲洗那一瞬间还是很酸爽的。如果你的两个窦腔都有炎症，那可能两边都需要穿刺。这个需要你签署相关说明，如果医生失误或者啥的，你是自愿的。当时我就有点犹豫，但想想自己太痛苦了，就签了。这个还是要看医生的水平，我做了两次，一次主任医师，一次普通医生，前者的水平很好，后者在我的提醒下才成功穿刺。主要是他想用锤子施压穿刺，认为脆骨太硬，不好穿透。我提醒他位置不对，才没有使用锤子。你说这能不害怕吗？万一真整不好，你的眼睛可能就废了。
  - 二是，这个穿刺的效果维持时间较短，窦腔可能会反复出现炎症。

## 物理治疗

- 去医院查找过敏原
  - 一是可以考虑避开过敏原
  - 二是可以参与舌下过敏原暴露疗法
- 买网上说的那个猪鼻子，或者胸前空气清新剂
  - 前者我看有人戴过，就是为了避免过敏原进入鼻腔，具体效果不清楚
  - 后者我同事有在用一个日本的牌子，1000 多吧，他说效果很好，相当于把你呼吸范围内的空气净化一下，也算是排除过敏原效果
- 去相对温暖潮湿的城市保养
  - 之前在深圳待过半年，感觉情况好很多
  - 其他沿海城市应该也是可以的，但我在山东烟台生活的那段时间，感觉效果不明显
- 如果在北方，保暖真的很重要，适当进行体育锻炼加强免疫力，严重鼻塞时，运动热身可以缓解
